# coop-tech TypeScript SDK

TypeScript packages for integrating with [coop.tech](https://coop.tech).

## Packages

| Package | Description |
|---------|-------------|
| [`@coop-tech/mcp-sidecar`](packages/mcp-sidecar) | Connect MCP servers to coop.tech as sidecars |

## Development

```bash
pnpm install
pnpm build
pnpm lint
```
